## Features

- Loading and preprocessing datasets using PyCaret
- Automated model comparison to identify the best-performing algorithm
- Hyperparameter tuning for model optimization
- Evaluation using cross-validation metrics
- Generating predictions on new/unseen data
- Visualization of model performance and results
